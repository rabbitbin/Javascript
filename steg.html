Steg för steg

Skissen med grundläggande idé

Baserande på planeringen hade jag börjat att göra och hade en tydlig bild på mig. När jag fortsatt projekt då förstått jag att det var lång ifrån min idé från början. Men jag vill fokusera på koder istället av design och försökte att använda nya teknik som jag lärde mig.


Att transfers värdet från en html sida till en annan html sida

Efter skissen hade jag skapat en enkel HTML sidan, en CSS fil och en Javascript fil samt kopplar till online url som behöver för webbplatsen. Sedan kopplar jag innehåll med mina tidigare äventyr spel till HTML fil och Javascript. Många ändringar och nya teknisk metoder krävs för att lyckas visa spelet som original. I bland annan försökt jag använda SVG som skapar en grafisk bakground med bild och färg. Och jag användar input fält och visar texter på HTML sidan med hjälpa av innerHTML istället av prompt och alert. Jag har fastnat länge för att hitta redan på hur kan jag transportera värdet från en HTML sida till den annan HTML sida. Det fanns flera olika sätt att göra, såsom skapar enkel server och användar PHP, men det var komplex och lite utanför kursen. Det finns metoden som använder formulär med metod "GET" och "POST" sedan man kan hämta värdet från annan sida med hjälpa av PHP. Till sist, vald jag att använda "localStorage" objekt i Javascript. Jag förstått inte direkt hur jag skriver och var jag lägger koder. Men det var så skönt att jag lyckades slutligen och det var bara två råder. Jag hittade bra lösning på EJS kapitel 18 - HTTP and forms.


Jämförelse mellan Canvas och SVG

Canvas är upplösningsberoende, du kan spara resultat bild som png eller jpg och det passar bra för grafiska interaktiv spel. Nackdelar är det stöd inte för "event handleds" och har dålig text renderings kapacitet. Däremot är SVG oberoende upplösning, stöd för "event handlers" och passar bra för applikation med stor renderings område såsom Google Maps. Nackdelar är det rendering är långsam om komplex och passar inte bra för spel applikationer.

Det tog jag mycket tid att få hitta en lösning och får något som jag vill men då fastnade att jag har svårt att hämta värdet från spelet och transfers eller anpassa det till "Time Machine".
